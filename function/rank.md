# ランクについて
経済鯖では、以下のようなランク制度を設けます。

ランクは、自動開放式にして、一部のランクは、条件に達したプレイヤーは自動的にランクを上昇させ、条件から外れたプレイヤーは、ランク減少とする。

## 使い方
1. `/rankcheck`を実行する。
2. 条件を確認する。
![11.png](img/11.png)
3. 条件に満たしていたら、ランクが付与される。
